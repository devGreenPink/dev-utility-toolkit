Library สำหรับจัดการ async data ด้วยแนวคิด stream (Observable) — ข้อมูลไหลมาเรื่อยๆ แทนที่จะมีแค่ครั้งเดียว เหมาะสำหรับ HTTP, DOM events, state management, และ real-time data

🚰 Array = น้ำในขวด (มีอยู่แล้ว)   |   Observable = ก๊อกน้ำ (ไหลมาเรื่อยๆ)
Observable
แหล่งข้อมูลที่ emit ค่าได้ — cold (ใหม่ทุก subscribe) หรือ hot (share กัน)
🚰 ก๊อกน้ำ
Observer
ผู้รับค่า — มี 3 callback: next, error, complete
🥤 แก้วน้ำ
Subscription
การเชื่อมต่อ Observable กับ Observer — เก็บไว้เพื่อ unsubscribe
🔌 เปิดก๊อก
Operator
ฟังก์ชันใน pipe() สำหรับแปลง, กรอง, รวม stream
⚙️ ตัวกรองน้ำ
Subject
Observable + Observer ในตัวเดียว — ส่งค่าเข้า stream จากภายนอกได้
🎛️ ก๊อกที่บังคับได้

Search Autocomplete
รอหยุดพิมพ์ 300ms → cancel request เก่า → search ใหม่
fromEvent(searchInput, 'input').pipe(
  map(e => e.target.value),
  debounceTime(300),
  distinctUntilChanged(),
  switchMap(term =>
    term ? http.get(`/api/search?q=${term}`) : EMPTY
  )
).subscribe(results => render(results));
Parallel API Calls
โหลดหลาย API พร้อมกัน รอทุกตัวเสร็จ
forkJoin({
  user: http.get('/api/user'),
  posts: http.get('/api/posts'),
  settings: http.get('/api/settings')
}).subscribe(({ user, posts, settings }) =>
  renderPage(user, posts, settings)
);
HTTP Retry + Timeout
retry 3 ครั้ง, หมดเวลา 5s, fallback gracefully
http.get('/api/data').pipe(
  timeout(5000),
  retry(3),
  catchError(err => of({ error: err.message })),
  finalize(() => hideSpinner())
).subscribe(data => render(data));
Angular Component Cleanup
หยุด subscription อัตโนมัติตอน component destroy
class MyComponent implements OnDestroy {
  private destroy$ = new Subject<void>();

  ngOnInit() {
    interval(1000).pipe(
      takeUntil(this.destroy$)
    ).subscribe(tick => update(tick));
  }

  ngOnDestroy() {
    this.destroy$.next();
    this.destroy$.complete();
  }
}

Simple State Store
BehaviorSubject เป็น lightweight global state
class Store {
  private state$ = new BehaviorSubject({ count: 0 });

  get count$() {
    return this.state$.pipe(
      map(s => s.count),
      distinctUntilChanged()
    );
  }

  increment() {
    const s = this.state$.getValue();
    this.state$.next({ ...s, count: s.count + 1 });
  }
}
